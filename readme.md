## Development setup

JS library: [solid-js](https://www.solidjs.com/)<br/>
CSS framework: [vanilla-extract](https://vanilla-extract.style/)<br/>

Provided scripts:

```
// Start local development
npm run start

// Lint the code
npm run lint:fix

// Transpile to javascript
npm run build

// Validate the extension
npm run ext:lint

// Build the extension
npm run ext:build
```
